# 🚀 go-journey

Repository ini berisi materi, contoh kode, dan proyek kecil untuk mempelajari dasar-dasar bahasa pemrograman Go (Golang).

- [x] Array, Slice, Map

- [x] Pointer

### ⚙️ journey 3: Fitur Khusus Go
